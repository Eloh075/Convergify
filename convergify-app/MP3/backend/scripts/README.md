# Backend Scripts

## Utility Scripts

### add_sample_jobs.py
Adds sample job postings to the database for testing.

### clear_empty_cache.py
Clears empty or invalid entries from the skill classification cache.

### extract_analysis_data.py
Extracts and displays analysis data from the database for debugging.

### generate_analysis_report.py
Generates a formatted analysis report from database results.

### list_available_models.py
Lists all available Gemini/Gemma models for the API keys.

## Usage

Run scripts from the backend directory:
```bash
cd Main_Project/backend
python scripts/script_name.py
```
